Цель проекта: Создание программно-аппаратного комплекса, который сни-
мает биосигналы человека (ЭМГ, ЭЭГ, PS, КГР), очищает их от шумов, вы-
деляет информативные признаки и с помощью искусственного интеллекта
определяет текущее функциональное состояние пользователя: Напряжение,
Утомление, Восстановление, Стресс, Перегрузка. По итогам анализа система
выдаёт человеку понятные рекомендации и (при необходимости) информа-
цию о подходящих лекарственных средствах.
                      Область применения:
• Самоконтроль состояния при умственной и физической работе (студен-
ты, офисные сотрудники, операторы).
• Спорт и реабилитация — отслеживание утомления и восстановления.
• Профилактика стресса и выгорания.
• Образовательные и исследовательские задачи в области биомедицин-
ской инженерии.
Аудитория:
• Студенты высших и средних специальных учебных заведений (особенно
в период сессии и подготовки к экзаменам).
• Офисные сотрудники (менеджеры, аналитики, бухгалтеры, IT-специалисты,
юристы).
• Спортсмены-любители (регулярно посещающие тренажерные залы, бе-
гуны, кроссфитеры).
• Спортсмены-профессионалы (члены команд, единоборцы, пловцы — для
предотвращения перетренированности).
             Установка и настройка ИИ-компонента
1. Требования к оборудованию: ПК, микроконтроллер с подключен-
ными 4 датчиками:ЭЭГ, ЭМГ,PS,КГР .
2. Необходимое ПО:
• ОС: Windows 10 / 11
• Python 3.10 или новее
• Chrome,Edge, Firefox
3. Шаги установки:
1.Установить Python 3.10+
2.Открыть командную строку (cmd) и установить необходимые библио-
теки
3.Обучить модель ИИ (выполняется один раз)
4.Подключить ESP32 к Wi-Fi и настроить отправку данных на ПК.
2.3 Запуск и управление ИИ
Пример команды: 1.
1.Запуск обучения модели (только при первом запуске или переобучении):
python step1 train v8 final.py
2.Запуск тестового режима без датчиков (для проверки модели): python
step2 test v8 full report.py
3.Запуск работы в реальном времени с датчиками (консоль): python step3
realtime v8 2.py
4.Запуск веб-сервера с ИИ (работа через сайт)
5.После запуска веб-сервера открыть в браузере.
Входные и выходные данные
• Вход: Векторы числовых значений с 4 датчиков:
– Пульс
– ЭМГ
– ЭЭГ
– КГР (уровень проводимости кожи / потоотделения).
• Выход:
– Классифицированное состояние (Напряжение», «Утомление», «Вос-
становление», «Стресс» или «Перегрузка».).
– Рекомендации и пример препарата для пользователя .
