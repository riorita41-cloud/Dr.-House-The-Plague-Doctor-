# 🧠 Dr.-House
[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)
[![Platform](https://img.shields.io/badge/platform-Windows%2010%2F11-lightgrey)](https://www.microsoft.com/windows)

**Программно-аппаратный комплекс** для анализа биосигналов человека (ЭМГ, ЭЭГ, пульс, КГР) с использованием искусственного интеллекта. Система определяет функциональное состояние (**Стресс**, **Утомление**, **Перегрузка**, **Напряжение**, **Восстановление**) и выдает персонализированные рекомендации, включая предложения по лекарственной поддержке.

## 🎯 Цель проекта

Создание программно-аппаратного комплекса, который:
- Снимает биосигналы человека (**ЭМГ**, **ЭЭГ**, **PS** (пульс), **КГР**)
- Очищает их от шумов и артефактов
- Выделяет информативные признаки
- С помощью **искусственного интеллекта** определяет текущее функциональное состояние пользователя
- Выдаёт понятные рекомендации и (при необходимости) информацию о подходящих лекарственных средствах

## 🚀 Возможности

- **4 канала сбора данных**: ЭЭГ, ЭМГ, Ps, кожно-гальваническая реакция (КГР)
- **Интеллектуальная фильтрация**: очистка биосигналов от артефактов и шумов
- **ИИ-классификатор** (обученная модель) — определяет 5 состояний:
  - 🟡 Напряжение
  - 🔵 Утомление
  - 🟢 Восстановление
  - 🔴 Стресс
  - 🟠 Перегрузка
- **Гибкий запуск**:  реальное время, веб-сервер
- **Рекомендательная система**: понятные советы + информация о лекарственных средствах (при необходимости)

## 🛠 Аппаратные требования

- **ПК** с ОС Windows 10 / 11
- **Микроконтроллер** (ESP32)
- **4 датчика**:
  - ЭЭГ (электроэнцефалограф)
  - ЭМГ (электромиограф)
  - ФПГ (пульс)
  - КГР (кожная проводимость / потение)
