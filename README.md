# 🧠 BioMind: Система психофизиологической диагностики с ИИ

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)
[![Platform](https://img.shields.io/badge/platform-Windows%2010%2F11-lightgrey)](https://www.microsoft.com/windows)

**Программно-аппаратный комплекс** для анализа биосигналов человека (ЭМГ, ЭЭГ, пульс, КГР) с использованием искусственного интеллекта. Система определяет функциональное состояние (**Стресс**, **Утомление**, **Перегрузка**, **Напряжение**, **Восстановление**) и выдает персонализированные рекомендации, включая предложения по лекарственной поддержке.

![Пример работы системы](https://via.placeholder.com/800x400?text=BioMind+Interface+Preview) <!-- Замените на реальный скриншот -->

## 📋 Область применения

| Сфера | Примеры использования |
|-------|----------------------|
| 🎓 **Образование** | Самоконтроль студентов в период сессии, исследовательские задачи |
| 💼 **Офис** | Профилактика выгорания у IT-специалистов, аналитиков, юристов |
| 🏋️ **Спорт** | Контроль восстановления и предотвращение перетренированности у бегунов, кроссфитеров, единоборцев |
| 🩺 **Реабилитация** | Отслеживание физического состояния в восстановительный период |

## 👥 Целевая аудитория

- Студенты вузов и колледжей
- Офисные сотрудники (менеджеры, бухгалтеры, разработчики)
- Спортсмены-любители и профессионалы
- Биомедицинские инженеры и исследователи

## 🚀 Возможности

- **4 канала сбора данных**: ЭЭГ, ЭМГ, фотоплетизмограмма (пульс), кожно-гальваническая реакция (КГР)
- **Интеллектуальная фильтрация**: очистка биосигналов от артефактов и шумов
- **ИИ-классификатор** (обученная модель) — определяет 5 состояний:
  - 🟡 Напряжение
  - 🔵 Утомление
  - 🟢 Восстановление
  - 🔴 Стресс
  - 🟠 Перегрузка
- **Гибкий запуск**: тестовый режим, реальное время, веб-сервер
- **Рекомендательная система**: понятные советы + информация о лекарственных средствах (при необходимости)

## 🛠 Аппаратные требования

- **ПК** с ОС Windows 10 / 11
- **Микроконтроллер** (ESP32)
- **4 датчика**:
  - ЭЭГ (электроэнцефалограф)
  - ЭМГ (электромиограф)
  - ФПГ (пульс)
  - КГР (кожная проводимость / потение)

## 💻 Программные требования

- ОС: Windows 10 / 11
- Python 3.10 или новее
- Браузер: Chrome, Edge, Firefox
- Wi-Fi доступ (для ESP32)
