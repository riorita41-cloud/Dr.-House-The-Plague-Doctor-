# 🧠 Dr. House — Чумной Доктор

[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)
[![Platform](https://img.shields.io/badge/platform-Linux%20%7C%20Windows-lightgrey)](https://www.microsoft.com/windows)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

**Программно-аппаратный комплекс** для анализа биосигналов человека (ЭМГ, ЭЭГ, пульс, КГР) с использованием искусственного интеллекта. Система определяет функциональное состояние (**Стресс**, **Утомление**, **Перегрузка**, **Напряжение**, **Восстановление**) и выдает персонализированные рекомендации, включая предложения по лекарственной поддержке.

---

## 🎯 Цель проекта

Создание программно-аппаратного комплекса, который:
- Снимает биосигналы человека (**ЭМГ**, **ЭЭГ**, **PS** (пульс), **КГР**)
- Очищает их от шумов и артефактов
- Выделяет информативные признаки
- С помощью **искусственного интеллекта** определяет текущее функциональное состояние пользователя
- Выдаёт понятные рекомендации и (при необходимости) информацию о подходящих лекарственных средствах

## 📋 Область применения

| Сфера | Примеры использования |
|-------|----------------------|
| 🎓 **Образование** | Самоконтроль студентов в период сессии, исследовательские задачи |

| 🩺 **Реабилитация** | Отслеживание физического состояния в восстановительный период |

## 🚀 Возможности

- **4 канала сбора данных**: ЭЭГ, ЭМГ, Ps, кожно-гальваническая реакция (КГР)
- **Интеллектуальная фильтрация**: очистка биосигналов от артефактов и шумов
- **ИИ-классификатор** (обученная модель RandomForest) — определяет 6 состояний:
  - 🟢 Норма
  - 🟡 Напряжение
  - 🔵 Утомление
  - 💜 Восстановление
  - 🟠 Стресс
  - 🔴 Перегрузка
- **Веб-интерфейс** с дашбордом, графиками и рецептурным бланком
- **Админ-панель** для управления обучением и сбором данных
- **Рекомендательная система**: понятные советы + информация о лекарственных средствах

## 🛠 Аппаратные требования

- **ПК** с ОС **Linux** (Ubuntu 22.04+, Arch, Fedora) или **Windows 10/11**
- **Микроконтроллер** ESP32 (для сбора реальных данных)
- **4 датчика**:
  - ЭЭГ (электроэнцефалограф)
  - ЭМГ (электромиограф)
  - ФПГ (пульс)
  - КГР (кожная проводимость / потение)

## 🚀 Запуск и управление

### Режимы работы

| № | Режим | Команда | Описание |
|---|-------|---------|----------|
| 1 | Обучение модели | `python step1_train_v8_final.py` | Только при первом запуске или переобучении |
| 2 | Тестовый режим | `python step2_test_v8_full_report.py` | Для проверки модели без датчиков |
| 3 | Реальное время (консоль) | `python step3_realtime_v8_2.py` | Работа с ESP32, вывод в консоль |
| 4 | Веб-сервер | `python server.py` | Запуск веб-интерфейса + API |

### Веб-интерфейс

После запуска сервера откройте в браузере: **http://localhost:5000**

**Страницы:**
- **Мониторинг** — графики пульса, ЭМГ, ЭЭГ в реальном времени
- **Снадобья** — ИИ-диагностика с рекомендациями и рецептурным бланком
- **История** — сохранённые сеансы
- **Профиль** — статистика и настройки
- **Админ-панель** — управление обучением, тестированием и сбором данных

### API Endpoints

| Метод | Путь | Описание |
|-------|------|----------|
| `POST` | `/api/analyze` | Отправить биосигналы, получить диагноз |
| `GET` | `/api/realtime` | Последние данные с ESP32 |
| `POST` | `/api/realtime/update` | Обновить данные с ESP32 |
| `GET` | `/api/admin/status` | Статус модели и системы |
| `GET` | `/api/admin/logs` | Лог операций |
| `POST` | `/api/admin/train` | Запустить обучение |
| `POST` | `/api/admin/test` | Запустить тестирование |
| `POST` | `/api/admin/realtime/start` | Запустить сбор с ESP32 |
| `POST` | `/api/admin/realtime/stop` | Остановить сбор с ESP32 |

## 📊 Входные и выходные данные

### Входные данные (векторы с 4 датчиков)

| Датчик | Сигнал |
|--------|--------|
| Пульс (PS) | Частота сердечных сокращений (уд/мин) |
| ЭМГ | Мышечная активность (%) |
| КГР | Кожная проводимость / потоотделение (%) |
| ЭЭГ | Электрическая активность мозга (%) |

### Выходные данные

**1. Классифицированное состояние:**
- 🟢 Норма
- 🟡 Напряжение
- 🔵 Утомление
- 💜 Восстановление
- 🟠 Стресс
- 🔴 Перегрузка

**2. Рекомендации для пользователя**

**3. Информация о лекарственных средствах** (при необходимости)

### Модель

- **Алгоритм**: RandomForest (600 деревьев, max_depth=14)
- **Точность**: 94.4%
- **Признаки**: mean, std, min, max, slope для каждого из 4 каналов (20 признаков)
- **Окно**: 20 точек
